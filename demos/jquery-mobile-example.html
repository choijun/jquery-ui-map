jQuery mobile with Google maps v3 demo

This is an example using the Google map version 3 jQuery UI plugin together with jQuery mobile framework. Since jquery mobile framework loads links through Ajax you either need to disable the jQuery Mobile's Ajax behavior or load the Google maps javascript in the 'real' first page. This example use ajax in navigation.

  • Basic map example
  • Basic map example with geolocation
  • Get directions example

To disable pages loading through Ajax bind 'mobileinit' and override $.mobile.ajaxEnabled. You need to bind 'mobileinit' before referencing the jQuery mobile script. 

						
							<script type="text/javascript">
								$(document).bind("mobileinit", function () {
									$.mobile.ajaxEnabled  = false;
								});
							</script>
							<script type="text/javascript" src="http://code.jquery.com/mobile/1.0a3/jquery.mobile-1.0a3.min.js"></script>
